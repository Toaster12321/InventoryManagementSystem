This project is designed to replicate an Inventory Management system used by major
retailers and suppliers to accurately store data inside of a database. It uses the windows form
application in visual studio connected to a MySQL database designed to track and store items,
suppliers, warehouses, orders, users and shipments. It is designed around the CRUD(create,
read, update and delete) operations allowing each respective table to utilize these tools when
necessary. This database supports one-to-many and many-to-many operations as well as an
easy to use interface for users. The backend of the system is connected to C# code that uses
the MySQL package to use procedures and queries created in the database.
